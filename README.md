# Node REST Server

This project is for implementation of a REST Server with ExpressJS.

## Usage

Set NodeJS version using `.nvmrc`

```
nvm use
```

Install dependencies

```
npm install
```

Run app in develop mode

```
npm run dev
```

Run app in production mode

```
npm start
```
